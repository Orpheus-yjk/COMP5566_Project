# [COMP]5566 Project

## 任务：在以太坊上检测蜜罐合约
> 
> (1)通过收集研究论文、报告和相关的博客，建立一个蜜罐合同的分类法。
> 
> (2)收集以太坊上的智能合约的源代码、ABI和字节码。
> 
> (3)创建一个工具或利用现有的现有工具来检测分类法中的每种契约。
